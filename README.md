# UNEC_FutureDev_Camp

# Gün 1 – HTML Əsasları

## 📌 Haqqında

Web Proqramlaşdırma Bootcamp-in ilk günündə HTML dilinin əsaslarını öyrəndik və sadə bir CV (Resume) veb səhifəsi hazırladıq.

Bu layihənin məqsədi HTML sənədinin strukturunu anlamaq və əsas HTML elementlərindən istifadə edərək səhifə yaratmaqdır.

---

## 📚 Öyrənilən Mövzular

- HTML sənədinin strukturu
- `<!DOCTYPE html>`
- `<html>`, `<head>` və `<body>` teqləri
- Başlıqlar (`<h1>` – `<h4>`)
- Paraqraflar (`<p>`)
- Keçidlər (`<a>`)
- Siyahılar (`<ul>` və `<li>`)
- Cədvəllər (`<table>`, `<tr>`, `<td>`)
- Sadə səhifə quruluşu
- `<center>` teqindən istifadə

---

## 💻 Hazırlanan Layihə

HTML istifadə edərək aşağıdakı bölmələrdən ibarət sadə CV səhifəsi hazırlandı:

- Naviqasiya menyusu
- Şəxsi məlumatlar
- Portfolio bölməsi
- Karyera nailiyyətləri
- İcma fəaliyyətləri
- Akademik kvalifikasiyalar
- Rəylər (Peer Reviews)

---

## 🛠 İstifadə Olunan Texnologiya

- HTML5

---

## 🎯 Öyrənilən Biliklər

Bu dərsin sonunda:

- HTML sənədinin əsas strukturunu qurmağı;
- Müxtəlif HTML teqlərindən istifadə etməyi;
- Başlıq, paraqraf, keçid, siyahı və cədvəl yaratmağı;
- Məzmunu bölmələr şəklində təşkil etməyi öyrəndik.
